# js-helper
k`s javascript helper/library/utils


### 依赖库
```shell
npm i --save core-js
npm i --save crypto-js
npm i --save lodash
npm i --save qs
```

### 安装测试库
```shell
# 官网 https://mochajs.org/
npm install --save-dev mocha
#或者
yarn add --dev mocha
mkdir test
```

### 执行测试
```shell
./node_modules/mocha/bin/mocha
#或者
npm test
```
